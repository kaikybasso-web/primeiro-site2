Bem-vindo ao meu site de exemplo Flebox

Este site ultiliza arquivo CSS separado, para criar o menu horizontal com Flexbox, além de uma aparência limpa e moderna

As tags estão todas presentes: <hearder>, <main>, <div>,(Container interno) <h1>, entre outros.

O menu horizontal é obtido com display: flex na lista <ul>. O CSS externo facilita a manutenção e o aprendizado separando da estrutura e da apresentação.

Agora o HTML cuida do conteúdo e o CSS do visual. Basta salvar dois arquivos e direcionar o link para o arquivo correto.
